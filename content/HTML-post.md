# Apa itu HTML?

**HTML (HyperText Markup Language)** adalah bahasa standar yang digunakan untuk membuat dan menyusun struktur halaman web. Bayangkan HTML sebagai kerangka tulang pada manusia; tanpa kerangka, kita tidak bisa membangun otot (CSS) atau otak (JS).

## Struktur Dasar HTML

Setiap file HTML memiliki struktur standar yang wajib ada. Berikut adalah contoh kerangka minimum sebuah halaman web:

```html
<!DOCTYPE html>
<html>
<head>
    <title>Halaman Pertama Saya</title>
</head>
<body>
    <h1>Halo Dunia!</h1>
    <p>Ini adalah paragraf pertama saya di web.</p>
</body>
</html>
```

## Elemen Penting dalam HTML

Ada beberapa elemen dasar yang akan sering kamu gunakan saat memulai pembelajaran:

1.  Headings: Digunakan untuk judul, mulai dari `<h1>` sampai `<h6>`.
2.  Paragraphs: Menggunakan tag `<p>` untuk membungkus teks.
3.  Links: Menggunakan tag `<a>` untuk menghubungkan antar halaman.
4.  Images: Menggunakan tag `<img>` untuk menampilkan gambar.

### Kenapa Harus Belajar HTML?

Memahami HTML adalah langkah pertama yang paling krusial sebelum kamu melangkah ke dunia CSS untuk desain atau Javascript untuk logika. Dengan HTML yang solid, struktur website kamu akan lebih mudah dipahami oleh mesin pencari (SEO) dan aksesibel bagi semua orang.

> "Struktur yang baik adalah awal dari kode yang hebat."
